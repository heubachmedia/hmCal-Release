## 4D Version & Platform

hmCal is available as a universal binary bundle for Macintosh and Windows. It was developed and tested with the current 4D versions 4D v16.3 and 4D v17. hmCal is only available for 4D v16 and above. hmCal supports the following operating systems: Mac 10.11-10.13, Windows 7 SP 1 and above.

<a id="nummer_00002"></a>

### Windows

hmCal 8 requires (at least) Direct X 11.1.

This is supported by Windows 7 SP 1 and Windows 2008 R2 SP1 with the platform update:

[https://support.microsoft.com/help/2670838/platform-update-for-windows-7-sp1-and-windows-server-2008-r2-sp1](https://support.microsoft.com/help/2670838/platform-update-for-windows-7-sp1-and-windows-server-2008-r2-sp1)

## Overview

hmCal allows you to add simply and quickly a complete calendar functionality into your 4D application. It is always guaranteed that functionality between Macintosh and Windows is identical.

The advantage of hmCal is that it can be integrated extremely simply into existing solutions. So appointments, virtual appointments (they do not have to exist in data records), repeated appointments etc. in complex situations (rights of access, on-the-fly-creation) can be represented and manipulated.

<a id="nummer_00005"></a>

## Features

- 9 views: day view, user day view (team calendar functionality), multi day view, user multi day view, project view, month view, resources view, year view and day user view
- Multi-day appointments
- Display up to 1000 days
- Complete adaptability of the calendar by using the programming language
- Shifting and extending dates by the user, drag & drop in all views
- User actions/events can be administered by programming language (callback method)
- Drag & drop of other form objects into the calendar
- Customizable colors, icons and tooltips
- Support of the iCalendar Standard /Import and export as iCalendar
- Multiple columns in user multi day view and project view
- Support of relations
- Support of repeated appointments
- Identical function range under Macintosh and Windows
- Offscreen drawing
- Export to different image formats (including SVG)
