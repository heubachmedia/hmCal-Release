# System processes

The organisation of system processes in hmCal can be use to find out, if the current 4D application is in the foreground or not. You can react on this behaviour and you can show messages to the user: [hmCal_DISPLAY NOTIFICATION](../../commands/system-processes/hmCal_DISPLAY-NOTIFICATION.md).

You also can bring the current application to the front: [hmCal_SET FRONT PROCESS](../../commands/system-processes/hmCal_SET-FRONT-PROCESS.md).

The system processes commands are very helpful, if the user should alarmed if an appointment is comming soon.
