# hmCal_SET TIMEINDICATOR TEXT

`hmCal_SET TIMEINDICATOR TEXT(area;text)`

| Parameter | Type | Direction | Description |
| --- | --- | --- | --- |
| area | Longint | -> | hmCal area |
| text | Text | -> | text  label |

<a id="nummer_00001"></a>

## Description

The command ***hmCal_SET TIMEINDICATOR TEXT*** sets a text label for the time indicator. This label is visable in the user multi day view only.
