# hmCal_SET LINE WIDTH

`hmCal_SET LINE WIDTH(area;selector;width)`

| Parameter | Type | Direction | Description |
| --- | --- | --- | --- |
| area | Longint | -> | hmCal area |
| selector | Longint | -> | line |
| width | Real | -> | width of a line |

<a id="nummer_00001"></a>

## Description

The command ***hmCal_SET LINE WIDTH*** sets the width of a line.

The following lines are supported:

- hmCal_clr_Hourline
- hmCal_clr_Halfhourline
- hmCal_clr_Dayline
- hmCal_clr_AppFrame
