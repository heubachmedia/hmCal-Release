# hmCal_mini_DISPLAY WEEK NUMBER

`hmCal_mini_DISPLAY WEEK NUMBER(area;display)`

```
Parameter          Type             Description
area               Longint      ->  hmCal-mini area
display            Longint      ->  1=display week-no
                                    2=don't display week-no
```

<a id="nummer_00001"></a>

## Description

The command ***hmCal_mini_DISPLAY WEEK NUMBER*** displays the week number in the calendar.

<a id="nummer_00002"></a>

## Example

The following example displays the week number:

```4d
hmCal_mini_DISPLAY WEEK NUMBER(hmCalmini;1)
```
