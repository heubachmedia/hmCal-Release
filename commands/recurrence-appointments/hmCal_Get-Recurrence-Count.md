# hmCal_Get Recurrence Count

`hmCal_Get Recurrence Count (area;reference) -> count`

| Parameter | Type | Direction | Description |
| --- | --- | --- | --- |
| area | Longint | -> | hmCal area |
| reference | Longint | -> | appointment reference |
| count | Longint | <- | occurrence |

<a id="nummer_00001"></a>

## Description

The command ***hmCal_Get Recurrence Count*** returns the count of the recurrence of the appointment defined by the parameter *reference*.

Also see: [hmCal_SET RECURRENCE COUNT](hmCal_SET-RECURRENCE-COUNT.md)
