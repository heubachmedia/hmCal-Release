# hmCal_SET RECURRENCE DATE

`hmCal_SET RECURRENCE DATE (area;reference;date)`

| Parameter | Type | Direction | Description |
| --- | --- | --- | --- |
| area | Longint | -> | hmCal area |
| reference | Longint | -> | appointment reference |
| date | Date | -> | End date |

<a id="nummer_00001"></a>

## Description

The command ***hmCal_SET RECURRENCE DATE*** sets the enddate of an repeated appointment. If you pass an empty date, the appointment is repeated endless.
