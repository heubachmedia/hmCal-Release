# hmCal_Get Recurrence Type

`hmCal_Get Recurrence Type (area;reference) -> type`

| Parameter | Type | Direction | Description |
| --- | --- | --- | --- |
| area | Longint | -> | hmCal area |
| reference | Longint | -> | appointment reference |
| type | Longint | <- | recurrence type |

<a id="nummer_00001"></a>

## Description

The command ***hmCal_Get Recurrence Type*** returns the recurrence type of the appointment, defined by the parameter *reference*.

Also see: [hmCal_SET RECURRENCE TYPE](hmCal_SET-RECURRENCE-TYPE.md)
