# hmCal_SECONDS2DATE

`hmCal_SECONDS2DATE(unix-time;date;seconds)`

| Parameter | Type | Direction | Description |
| --- | --- | --- | --- |
| unix-time | Longint | -> | unix-time |
| date | Date | <- | date |
| seconds | Longint | <- | seconds |

<a id="nummer_00001"></a>

## Description

The command ***hmCal_SECONDS2DATE*** returns the date and time of the given unixtime. The unix-time is the count of seconds since january 1st, 1970.
